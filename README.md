# Iris Flower Classification

## Project Overview
This project uses the Iris flower dataset to classify flowers into three species:
- Setosa
- Versicolor
- Virginica

## Technologies Used
- Python
- Scikit-learn
- NumPy
- Matplotlib

## Machine Learning Algorithm
- Logistic Regression

## Dataset
The Iris dataset is loaded using Scikit-learn's built-in `load_iris()` function.

## Steps Performed
1. Loaded the Iris dataset.
2. Split the dataset into training and testing data.
3. Trained a Logistic Regression model.
4. Predicted flower species.
5. Evaluated the model using accuracy and classification report.

## Result
- Model Accuracy: **100%**
- Sample Prediction: **Setosa**
